# correcao-lexical-prematura-llms-multimodais
Estudo de caso sobre correção lexical prematura em LLMs multimodais. Documenta falha em Claude, Meta AI, Gemini e ChatGPT ao corrigir 'subscreveu'→'subiu' sem contexto de imagem. Propõe: correção com consciência de incerteza, julgamento diferido e calibração lexical. Inclui testes, PDF com prints e relatório técnico.
